===========================
Kaprien Server Installation
===========================

Kaprien Server is a combination of micro-services.
 - ``kaprien-rest-api``
 - ``kaprien-repo-worker``

Both services require a Broker server, Redis server and a Web server to publish the
signed Repository Metadata.
